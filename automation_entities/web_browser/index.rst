===========
web_browser
===========

.. module:: automation_entities.web_browser

The :ref:`automation_entities-web_browser` module forms the basis of any
:class:`automation_entities.entities.Entity` that needs to interact with a web
page via a browser. It wraps selenium but provdes some niceties on top of it
that can be used to better automate interactions. The main class to instantiate
is the :class:`WebBrowser` class.
